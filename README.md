# carrera-sistemas-docs-operativos

Repositorio de documentación operativa de la Carrera de Sistemas.

Este repositorio agrupa la documentación de trabajo cotidiano que guía la ejecución académica y técnica de la carrera. No reemplaza la documentación oficial del programa; la complementa con lineamientos operativos, guías, estándares y plantillas para el trabajo diario.

## Qué va en este repo (operativo)

En `carrera-sistemas-docs-operativos` se publica contenido práctico y de uso diario para estudiantes y docentes:

- guías operativas paso a paso
- checklists y criterios de entrega
- plantillas reutilizables
- resúmenes operativos de documentos oficiales (sin sustituir la fuente oficial)

## Reglas de uso

- La documentación oficial del programa vive en su repositorio propio y no debe duplicarse aquí.
- Los estándares operativos de software deben mantenerse fuera de los microservicios.
- Cada cambio relevante debe entrar mediante PR con descripción clara del objetivo y del impacto.
- Los repositorios de microservicios deben referenciar este repo, no copiar estos documentos.
