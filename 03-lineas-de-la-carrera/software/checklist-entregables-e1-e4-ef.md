# Checklist Artefactos E1-E4 + EF — Línea Software

## Checklist general

- [ ] Problema y alcance definidos.
- [ ] Requerimientos priorizados y trazables.
- [ ] Diseño técnico consistente con el problema.
- [ ] Código versionado y ordenado.
- [ ] Pruebas y resultados documentados.
- [ ] Sustentación técnica preparada.

## Validación por etapa

### E1
- [ ] Contexto, problema y objetivos claros.
- [ ] Requerimientos funcionales/no funcionales definidos.

### E2
- [ ] Arquitectura y modelo técnico documentados.
- [ ] Decisiones técnicas justificadas.

### E3
- [ ] Implementación funcional verificable.
- [ ] Evidencia de pruebas iniciales.

### E4
- [ ] Validaciones finales ejecutadas.
- [ ] Hallazgos y ajustes registrados.

### EF
- [ ] Integración de artefactos `E1-E4` completa.
- [ ] Sustentación técnica y funcional lista.
- [ ] Lecciones aprendidas consolidadas.
